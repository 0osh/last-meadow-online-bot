# last-meadow-online-bot
simple script that plays the last meadow online game automatically from discords 2026 april fools event

## how to use
copy the contents of [the javascript file](last-meadow-online-bot.js) into the console (google if you dont know how to access the console in discord) and open the game. then just choose what you want to automate.

## current features
- spams adventure button
- completes arrow key crafting task
- completes the shooting down targets task for archers
- ui to configure what gets spammed/automated
